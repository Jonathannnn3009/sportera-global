Manage Age Categories per Tournament

Set the exact age categories available for each tournament, along with a cutoff date for each — a player only qualifies for a category if they were born on or after that date (i.e. not overage). The registration form checks this automatically and blocks submission if the player doesn't qualify for the age category they'd fall under.

Cutoff date is optional — leave blank if you don't want DOB validation for that category. A player born before this date is overage and cannot register under it.

Manage Pools per Tournament

If a tournament has enough teams, split them into pools (Pool A, Pool B, or any name you like). Add as many as you need, rename or remove them any time — Manage Teams, Add Fixture, and the League Table all follow whatever pools exist here.

Bulk Upload Fixtures (Excel)

Upload an .xlsx file with one row per fixture. Team, Tournament, and Age Category names must match exactly what's already registered (case-sensitive) — download the template below and fill it in, or export it and hand it to whoever is scheduling matches.

Columns (first row = headers, exact spelling):
Tournament — exact tournament name, e.g. "Football 5 Premier League"
City — leave blank unless this is a multi-city tournament (e.g. IF5L); must match a city already configured for it
Age Category — e.g. "U8" — must be one this tournament actually has
Pool — optional; leave blank for no pool
Home Team / Away Team — exact registered team names
Date — YYYY-MM-DD, e.g. 2026-08-15
Time — 24-hour HH:MM, e.g. 16:30
Venue — optional
Ground — optional; ground/pitch number if the venue has more than one, e.g. "Ground 1"
Round — optional; one of League Stage, Quarter-Final, Semi-Final, Third Place, Grand Final (defaults to League Stage if blank or unrecognized)
